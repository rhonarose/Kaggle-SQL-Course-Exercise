# Kaggle SQL Course Exercises

This repository includes the notebooks from the Kaggle SQL Course, designed to help you learn and practice essential SQL concepts. Each notebook focuses on specific topics, with exercises and solutions that demonstrate how to query databases efficiently.

## Topics Covered
- SQL Basics: `SELECT` statements, `WHERE` clauses
- `JOINs`: Combining data from multiple tables
- Aggregation: `GROUP BY`, `HAVING`
- Sorting & Filtering: `ORDER BY`, `LIMIT`
- Analytic Functions: `WINDOW` functions, `RANK`, `ROW_NUMBER`
- Subqueries: Nested queries for advanced data retrieval
